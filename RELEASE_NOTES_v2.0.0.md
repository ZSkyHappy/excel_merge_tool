# Excel 批量合并工具 v2.0.0

## 下载

下载 `ExcelMergeTool-Windows-x64.exe` 后双击运行，无需安装 Python。可使用 `SHA256SUMS.txt` 校验文件完整性。

## 主要更新

- 保留经典 A:Q 合并模式。
- 新增精确文件预览和逐文件工作表、表头行、CSV 参数设置。
- 新增自定义字段映射、必填/可选字段及别名匹配。
- 新增可导入、导出的映射方案。
- 新增 `.xlsm` 和 CSV 输入。
- 新增多个文件级特殊字段和可配置标签模板。
- 普通处理报告升级为带筛选和颜色提示的 Excel 工作簿。
- 继续使用流式读写、自动分卷、取消清理和事务式结果发布。

## 运行环境

- Windows 11 64 位：已验证。
- Windows 10 64 位：未验证。
- 输入格式：`.xlsx`、`.xlsm`、`.csv`。
- 输出格式：`.xlsx`。

## 已知限制

- 实际业务文件兼容性未验证，首次使用请先备份并抽样核对。
- XLSM 宏不会执行或复制。
- 公式只读取源文件上次保存的缓存结果；未计算公式可能为空。
- 暂不支持旧版 `.xls`、源样式、合并单元格、去重、排序和汇总。
- EXE 暂未进行数字签名，Windows 可能显示 SmartScreen 提示。
